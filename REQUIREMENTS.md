## 二、数据库重构（⬜ 待确认字段后执行）

### 2.1 架构变化说明

| 变化点 | 旧方案 | 新方案 |
|--------|--------|--------|
| 订单来源 | 仅堂食 | 堂食 + 美团外卖 + 饿了么 + 京东到家 |
| orders 表 | 无渠道字段 | 新增 `channel`（渠道）、`platform`（平台）字段 |
| 统计表 | 不区分渠道 | 按渠道分开统计，支持多维度筛选 |

### 2.2 新数据库表设计（草案，待实际字段确认后定稿）

#### orders（全渠道订单主表）
```
新增字段：
  channel      TEXT   -- 渠道：堂食 / 外卖
  platform     TEXT   -- 平台：美团 / 饿了么 / 京东到家 / 堂食
  delivery_fee REAL   -- 配送费（外卖专属，堂食为0）
```
> 其余字段与现有 orders 表保持一致

#### order_dishes（菜品明细）
```
现有字段保持，来源改为"菜品销售明细"Excel
新增字段（如有）：
  business_date TEXT  -- 营业日期（便于直接按天查询）
```

#### order_payments（收款明细）
```
现有字段保持，来源改为"收款明细"Excel
待确认字段是否有差异
```

#### sales_summary（日度统计）—— 按渠道拆分
```
新增字段：
  channel TEXT  -- 堂食 / 外卖 / 全渠道
  -- 唯一约束改为 (store_id, date, channel)
```

#### monthly_summary（月度统计）—— 按渠道拆分
```
新增字段：
  channel TEXT  -- 堂食 / 外卖 / 全渠道
  -- 唯一约束改为 (store_id, month, channel)
```

#### item_sales_summary（菜品统计）
```
现有结构保持（菜品数据不区分渠道）
```

### 2.3 数据迁移策略
- 现有数据（堂食）全部清空，重新从新格式 Excel 导入
- 历史堂食数据改用全渠道 Excel 重新下载导入

### 页面1：集团驾驶舱

**路由**：`/`（默认首页）

**筛选器**
- 月份选择器（默认当月）
- 门店多选下拉（默认全部门店）
- 渠道切换：全渠道 / 堂食 / 外卖（新增）

**KPI 卡片区**（一行横排）

| 卡片 | 指标 | 环比显示 | 数据来源 |
|------|------|----------|----------|
| 总营业额 | `total_revenue` 合计 | ✅ 涨跌箭头+% | monthly_summary |
| 总实收 | `total_sales` 合计 | ✅ | monthly_summary |
| 总订单数 | `order_count` 合计 | ✅ | monthly_summary |
| 平均客单价 | 加权平均 | ✅ | monthly_summary |
| 优惠率 | 优惠/营业额 % | ✅ | monthly_summary |
| 毛利率 | — | ⬜ 预留（待成本数据） | — |

**图表区（左右布局）**

- 左侧：**营收趋势折线图**（按天，切换：近7天 / 近30天 / 本月）
  - 多线：堂食营收 + 外卖营收（可叠加显示）

- 右侧：**门店营收排名**（横向柱状图）
  - 按当月总营业额降序，点击跳转单店分析

**数据表**：本月 vs 上月各门店对比

| 列 | 字段 |
|----|------|
| 门店名称 | store_name |
| 上月营业额 | prev total_revenue |
| 本月营业额 | curr total_revenue |
| 营业额环比 | % 红/绿色标注 |
| 上月订单数 | prev order_count |
| 本月订单数 | curr order_count |
| 订单环比 | % |
| 上月客单价 | prev avg_order_amount |
| 本月客单价 | curr avg_order_amount |
| 客单价环比 | % |

**图表区（上下布局）**
- 上方：**营收趋势折线图**（双线：营业额 + 实收）

**数据表**：每日 / 每月明细

| 列 | 字段 |
|----|------|
| 日期 / 月份 | date / month |
| 营业额 | total_revenue |
| 实收 | total_sales |
| 优惠额 | total_discount |
| 优惠率 | discount_ratio % |
| 订单数 | order_count |
| 客单价 | avg_order_amount |
