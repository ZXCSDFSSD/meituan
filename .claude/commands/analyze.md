# /analyze — 数据探索性分析（EDA）报告

对美团餐饮数据系统中的 **Excel 文件** 或 **SQLite 数据库** 生成完整的探索性数据分析报告。

## 用法

```
/analyze                          # 分析 data/meituan.db 全库
/analyze data/meituan.db          # 分析 SQLite 数据库（全库）
/analyze data/meituan.db orders   # 分析指定表
/analyze <Excel文件路径>           # 分析 Excel 文件
```

## 参数说明

`$ARGUMENTS` 为用户传入的参数，格式：
- 空（无参数）→ 分析 `data/meituan.db` 全库
- `data/meituan.db` → 全库分析
- `data/meituan.db <表名>` → 单表深度分析
- `downloads/.../xxx.xlsx` → Excel 文件分析

---

请执行以下步骤，生成完整 EDA 报告：

**第1步：运行分析脚本**

根据参数 `$ARGUMENTS` 决定运行哪个命令：
- 若参数为空或为 `--all`：运行 `node scripts/analyze_data.js --all`
- 若参数为 `.db` 文件 + 表名：运行 `node scripts/analyze_data.js <db路径> <表名>`
- 若参数为 `.db` 文件：运行 `node scripts/analyze_data.js <db路径>`
- 若参数为 `.xlsx` 文件：运行 `node scripts/analyze_data.js <xlsx路径>`

使用 Bash 工具在项目根目录执行，捕获完整输出。
