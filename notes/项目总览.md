## 系统架构

```
美团后台（网页）
    │  Puppeteer 自动化下载（downloader）
    ▼
downloads/
├── 全渠道订单明细/
├── 菜品销售明细/
└── 收款明细/
    │  Excel 解析 + 写入（storage）
    ▼
data/meituan.db（SQLite）
    │  REST API（api）
    ▼
前端 Dashboard（待开发）
```

## 数据库（v3，10张表）

| 表名 | 来源 | 说明 |
|------|------|------|
| stores | 菜品销售明细 | 8家门店，含 location_type/has_partner |
| orders | 全渠道订单明细 | 订单主表 |
| order_dishes | 菜品销售明细 | 品项明细，含点菜时间/菜品小类 |
| order_payments | 收款明细 | 目前未写入 |
| sales_summary | 聚合自 orders | 日度统计（含渠道拆分） |
| monthly_summary | 聚合自 orders | 月度统计（含渠道拆分） |
| item_sales_summary | 菜品销售明细 | 品项日度汇总 |
| payment_method_summary | 收款明细 | 收款方式汇总 |
| **timeslot_summary** | 品项明细点菜时间 | **24小时时段统计（v3新增）** |
| **category_mapping** | 手工维护 | **品类名称标准化映射（v3新增）** |
